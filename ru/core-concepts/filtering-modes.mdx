## Обзор режимов фильтрации

JMP предоставляет два режима фильтрации задач: "Activity" и "Columns". Выбор режима осуществляется через параметр "Filtering Mode" в настройках отчета.

## Режим "Activity"

В режиме "Activity" учитываются все задачи, которые были активны (т.е. находились в любой из выбранных колонок) в течение заданного временного промежутка.

### Особенности режима "Activity":

- Охватывает все задачи, которые были в работе в выбранный период.
- Подходит для анализа общей активности на доске.
- Позволяет увидеть полную картину движения задач по доске.

## Режим "Columns"

Режим "Columns" фильтрует задачи на основе их прохождения через указанные колонки завершения (Completion Columns) в заданном временном промежутке.

### Особенности режима "Columns":

- Фокусируется на конкретных этапах процесса.
- Позволяет анализировать время прохождения задач через определенные стадии.
- При выборе этого режима активируется дополнительное поле "Completion Columns", где можно указать одну или несколько колонок для анализа.

### Логика работы режима "Columns":

1. Задача учитывается, если она попала хотя бы в одну из указанных Completion Columns в заданном временном промежутке.
2. Используется самая поздняя дата попадания задачи в любую из Completion Columns.
3. Если после Completion Column был возврат в предыдущую колонку, то задача не учитывается в выборке.

## Применение режимов фильтрации

Выбор режима фильтрации влияет на следующие отчеты:

- Cycle Time
- Aging Chart
- Predictability
- Tasks Table

Это обеспечивает единый подход к анализу данных во всех отчетах.

## Рекомендации по выбору режима

- Используйте режим "Activity" для общего анализа процесса и выявления узких мест на всех этапах работы.
- Применяйте режим "Columns" для детального анализа конкретных этапов процесса, например, времени разработки или тестирования.

## Заключение

Правильный выбор режима фильтрации позволяет получить наиболее релевантные данные для вашего анализа. Экспериментируйте с обоими режимами, чтобы лучше понять, какой из них наиболее эффективен для ваших конкретных задач и процессов.
