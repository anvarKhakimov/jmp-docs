---

## Открытие JMP

1. Откройте вашу доску в Jira.
2. Найдите одну из кнопок JMP:
   - Кнопка "Analyze Metrics" в верхнем правом углу доски.
   - Кнопка "Metrics" в левом боковом меню.
3. Нажмите на любую из этих кнопок, чтобы открыть интерфейс JMP.

## Выбор отчета

В интерфейсе JMP вы увидите несколько типов отчетов:

- Cycle Time
- Throughput Chart
- Aging Chart
- Cumulative Flow Diagram
- Predictability
- Task Table

Выберите интересующий вас отчет, кликнув на соответствующую вкладку.

## Настройка фильтров

Перед генерацией отчета вы можете настроить фильтры:

1. Выберите временной период для анализа.
2. Укажите нужные колонки доски.
3. При необходимости, выберите конкретные плавательные дорожки или быстрые фильтры.

## Анализ данных

Изучите сгенерированный отчет. Обратите внимание на:

- Общие тренды
- Аномалии или выбросы
- Соответствие ожиданиям и целевым показателям

## Действия на основе данных

На основе полученной информации вы можете:

- Определить области для улучшения
- Выявить узкие места в процессах
- Принять обоснованные решения по оптимизации рабочего процесса

## Дальнейшие шаги

Для более глубокого понимания функциональности JMP и интерпретации отчетов, рекомендуем ознакомиться с разделами документации, посвященными каждому типу отчета.

Не забывайте регулярно проводить анализ с помощью JMP для отслеживания прогресса и выявления новых возможностей для улучшения процессов.
