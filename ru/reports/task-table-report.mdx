## Обзор

Tasks Table предоставляет детальное табличное представление задач вашего проекта. Этот отчет позволяет анализировать задачи, попадающие под выбранные фильтры, с возможностью сортировки и просмотра временной шкалы каждой задачи.

## Основные функции

### Фильтрация задач

Для настройки выборки задач используются те же фильтры, что и в других отчетах. Подробнее о работе с фильтрами можно узнать в разделе [Выборка данных и фильтрация](/ru/docs/core-concepts/data-sampling-and-filtering).

### Настройка формата отображения времени

Вы можете выбрать удобный формат отображения времени:

- Часы
- Дни
- Недели
- Временная метка (timestamp)

Примеры отображения для задачи, находившейся в работе 10 дней и 8 часов:

- Часы: "248 hours"
- Дни: "10.33 days"
- Недели: "1.48 weeks"
- Временная метка: "10d 8h"

Изменение формата времени динамически обновляет данные в таблице.

### Экспорт данных

Кнопка "Download CSV" позволяет экспортировать все отфильтрованные задачи в CSV-файл для дальнейшего анализа.

Важно отметить, что в CSV-файле форматы отображения времени "часы", "дни" и "недели" выводятся как числовые значения без указания единиц измерения. Например:

- 248 (для часов)
- 10.33 (для дней)
- 1.48 (для недель)

Формат "временная метка" (timestamp) в CSV сохраняется в том же виде, что и в таблице.

### Сортировка

Возможна сортировка по любому столбцу таблицы, включая ID задачи, общее время цикла и время в каждой колонке.

## Структура таблицы

### Заголовок таблицы

- **ID**: Уникальный идентификатор задачи, кликабельная ссылка на задачу в Jira.
- **Σ Lead Time**: Общее время выполнения задачи.
- **Колонки процесса**: Время, проведенное задачей в каждой выбранной колонке рабочего процесса.

### Строки задач

Каждая строка представляет отдельную задачу и содержит:

- Возможность раскрытия для просмотра детальной временной шкалы.
- ID задачи с прямой ссылкой на Jira.
- Общее время выполнения задачи.
- Время, проведенное в каждой выбранной колонке процесса.

## Детальный просмотр задачи

При раскрытии строки задачи отображается:

### Временная шкала (Timeline)

- Визуальное представление жизненного цикла задачи.
- Каждый сегмент представляет время, проведенное в определенной колонке.
- Цветовая кодировка сегментов соответствует различным этапам процесса.

### Интерактивные элементы

- При наведении на сегмент временной шкалы отображается всплывающая подсказка с названием колонки и точным временем, проведенным в ней.

## Интерпретация данных

- **Общее время цикла**: Позволяет оценить общую продолжительность выполнения задачи.
- **Время в колонках**: Помогает выявить этапы, на которых задачи задерживаются дольше всего.
- **Временная шкала**: Дает визуальное представление о прохождении задачи через различные этапы процесса.

## Практическое применение

1. **Анализ эффективности процесса**: Используйте сортировку по общему времени цикла для выявления самых длительных задач.
2. **Выявление узких мест**: Анализируйте время в отдельных колонках для определения этапов, требующих оптимизации.
3. **Сравнение задач**: Используйте временные шкалы для сравнения жизненных циклов разных задач.
4. **Отчетность**: Экспортируйте данные в CSV для создания детальных отчетов и презентаций.

## Заключение

Tasks Table предоставляет детальный взгляд на каждую задачу в вашем рабочем процессе. С помощью этого отчета вы можете анализировать эффективность вашего процесса разработки, выявлять проблемные области и принимать обоснованные решения для оптимизации рабочего потока.
