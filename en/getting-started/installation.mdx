## System Requirements

- Chromium-based browser (Chrome, Edge, Yandex, Opera, etc.)
- Access to Jira

## Installation Process

1. Go to the [Chrome Web Store](https://chromewebstore.google.com/detail/jira-metrics-plugin/ehhncnkhbchjllmnbcfebjllhnkgnfpc).

2. On the extension page, click the "Install" button.

   ![Installing the extension](/images/install-extension.png)

3. Confirm the installation if your browser asks for permission.

4. After the installation is complete, you'll see the JMP icon in your browser's extension panel.

## Verifying the Installation

To make sure JMP is installed correctly:

1. Open Jira in your browser.
2. Go to any task board.
3. You should see two new buttons:
   - An "Analyze Metrics" button in the top right corner of the board.
   - A "Metrics" button in the left sidebar menu.

   ![JMP buttons on Jira board](/images/jmp-buttons.png)

If you can see these buttons, JMP is successfully installed and ready to use!

## Troubleshooting

If you don't see the JMP buttons after installation:

1. Make sure the extension is enabled in your browser settings.
2. Try refreshing the Jira page.
3. If the problem persists, try disabling and re-enabling the extension.

If you experience any other installation issues, please contact our [Telegram support chat](https://t.me/JiraMetrics).
